Update ist auf diesem System nicht aktiviert

Damit du Updates aus dem Browser starten kannst, muss die Umgebungsvariable KEEPUP_UPDATE_TOKEN im keepup.service gesetzt sein.

Beispiel (systemd Override):

systemctl edit keepup.service

[Service] Environment=KEEPUP_UPDATE_TOKEN=DEIN_TOKEN

Danach:

systemctl daemon-reload systemctl restart keepup.service

Update ist auf diesem System aktiviert und kann direkt aus dem Browser gestartet werden.
